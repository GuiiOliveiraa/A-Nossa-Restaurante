# 🍽️ A Nossa Restaurante - Cardápio Digital

## 📋 Sobre o Projeto

O **A Nossa Restaurante - Cardápio Digital** é um projeto desenvolvido com o objetivo de modernizar o atendimento do restaurante, oferecendo uma experiência prática e intuitiva para os clientes através de um sistema de pedidos on-line.

A plataforma foi projetada com foco em dispositivos móveis, permitindo que os usuários visualizem o cardápio, adicionem produtos ao carrinho e finalizem pedidos diretamente pelo WhatsApp.

Além disso, o sistema prevê uma área administrativa para gerenciamento de pedidos e manutenção da plataforma.

---

# 🚀 Funcionalidades

- 📖 **Cardápio On-line**
  - Visualização de pratos, bebidas e sobremesas
  - Exibição de preços e descrições dos produtos

- 🛒 **Sistema de Carrinho**
  - Adição e remoção de itens
  - Controle de quantidade dos produtos

- 📲 **Integração com WhatsApp**
  - Finalização do pedido via WhatsApp
  - Envio automático das informações do pedido

- 🚚 **Gestão de Delivery**
  - Coleta de nome e endereço do cliente
  - Organização de informações para entrega

- ⚙️ **Painel Administrativo**
  - Gerenciamento interno da plataforma
  - Controle de pedidos e manutenção do sistema
